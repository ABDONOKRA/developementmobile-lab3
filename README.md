# Lab 2 : Formulaire et Intent Explicite - Android

Ce projet est un laboratoire pratique pour apprendre les bases de la navigation entre activités et la transmission de données dans Android.

## 📱 Aperçu
L'application se compose de deux écrans :
1.  **Écran de Formulaire** : Permet à l'utilisateur de saisir son Nom, Email, Téléphone, Adresse et Ville.
2.  **Écran Récapitulatif** : Affiche les données saisies par l'utilisateur.

## ✨ Fonctionnalités
- ✅ Saisie de données avec différents types de claviers (`inputType`).
- ✅ Validation simple des champs obligatoires (Nom et Email).
- ✅ Utilisation d'un **Intent explicite** pour naviguer vers un second écran.
- ✅ Transmission de données via les `Extras`.
- ✅ Mise en page avec `ScrollView`, `LinearLayout` et `ConstraintLayout`.

## 🛠️ Technologies Utilisées
- **Langage** : Java
- **UI** : XML Layouts
- **Composants** : `EditText`, `Button`, `TextView`, `ScrollView`, `Intent`, `Toast`.

## 📂 Structure du Projet
- `MainActivity.java` : Gère la saisie et l'envoi des données.
- `Screen2Activity.java` : Affiche le récapitulatif.
- `activity_main.xml` : Design du formulaire.
- `activity_screen2.xml` : Design de l'écran de récapitulatif.
